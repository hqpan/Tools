# 1. 面试题

- Git 的工作区：即存放`.git`文件的文件夹；
- 版本库：`.git`文件，包括 Stage 暂存区和分支；
- Git 的工作流？
  - Stage 暂存区：`git add filename`，将修改添加到暂存区；
  - History 版本库：`git commit`，将暂存区的修改提交到当前分支；
  - 远程库：`git push`，将修改推送到远程库；
